DESARROLLO WEB

HTML,CSS y Javascript


Son los tres pilares fundamentales del desarrollo web que trabaja juntos para crear páginas web dinámicas interactivas.

HTML se encarga de las estructuras y el contenido,CSS del diseño y presentación y Javascript de la interactividad y la funcionalidad.


Definiciones:


HTML (Hyper Text Makeup Language):

Es el lenguaje marcado que define la estructura y el contenido de una página web. Utiliza etiquetas para demilitar diferentes elementos, como párrafos,encabezados,imágenes,enlaces,etc.

CSS (Cascading Style Sheets):

Es un lenguaje de estilos que se usa para controlar la presentación de una página web. Permite definir aspectos como los colores,fuentes,tamaños,márgenes,etc, de los elementos HTML.

Javascript (Program Language):

Es en lenguaje de programación que permite agregar funcionalidad interactiva a una página web. Permite manipular elementos HTML,responder a eventos del usuario,realizar cálculos,etc.


Analogia de la web y una casa:

HTML

Estructura de la casa


Es como los cimientos, las paredes, techos y distribuciones de las habitaciones.


HTML define la estructura básica de una página web: qué elementos hay (título,párrafos,botones,imágenes,etc), pero sin preocuparse por como se ven

CSS

Decoración y estilo de la casa.


Es la pintura de las paredes, el tipo de piso, los muebles, las cortinas, el diseño de la cocina


CSS se encarga de como se ve la página: colores, fuentes, márgenes,alíneaciones, estilos, responsivos... lo visual.

Javascript

Los sistemas electricos y electronicos de la casa.


Es como los interruptores que encienden luces, las puertas automaticas, el termostato inteligete.


Javascript añade funcionalidad e interactividad haciendo que pasen cosas como respuesta a un evento.

Resumen

Los tres lenguajes escenciales para el desarrollo web y cada uno juega un papel crucial en la creación de páginas web modernas y funcionales.
